Breaking Bad Guess Quotes

BreakingQuotes is a personal project that I built for learning purposes. It comprises a web platform where you can test your knowledge about the Breaking Bad series by trying to figure out which character said which quotes. The more correct answers you have, the higher you will be on the scoreboard.

Open API

The project includes an open API, if you want to use it for your own projects. The endpoints are:

  • GET api/characters/

    Retrieve a list of all characters.

    http://breakingquotes.com/api/characters/
    [ { "id": 1, "name": "Walter White" }, { "id": 2, "name": "Jesse Pinkman" }, { "id": 3, "name": "Saul Goodman" } ]
  • GET api/characters/{id}

    Retrieve a specific character by ID.

    http://breakingquotes.com/api/characters/{id}
    { "id": 1, "name": "Walter White" }
  • GET api/quotes/

    Retrieve a list of all quotes.

    http://breakingquotes.com/api/quotes/
    [ { "id": 1, "text": "I am the one who knocks!", "character": { "id": 1, "name": "Walter White" } }, { "id": 2, "text": "Yeah, science!", "character": { "id": 2, "name": "Jesse Pinkman" } } ]
  • GET api/quotes/{id}

    Retrieve a specific quote by ID.

    http://breakingquotes.com/api/quotes/{id}
    { "id": 1, "text": "I am the one who knocks!", "character": { "id": 1, "name": "Walter White" } }
  • GET api/quotes/random

    Retrieve 20 random quotes.

    http://breakingquotes.com/api/quotes/random
    { "id": 2, "text": "Yeah, science!", "character": { "id": 2, "name": "Jesse Pinkman" } }
